# Character Settings Prompt v3.3 Evaluation Draft

**Version:** 3.3
**Date:** 2026-01-22
**Target Model:** Gemma 3 12B / Gemma 2 27B
**Base Concept:** Hybrid of "Simple Description" (Rich Context) and "JSON Structure" (Strict Logic).

---

## 1. 変更点の概要

以前の評価レポート（Phase 0.2）の結果に基づき、以下の改良を行いました。

1.  **JSON Schemaへの完全移行**:
    * Gemma 3等の最新モデルが最も理解しやすい構造化データ形式を採用。
    * ただし、値（Value）の中身は「Simple形式」のように**文章で詳細に記述**し、表現力の低下を防ぐ。
2.  **思考プロセス（Thought）の強制**:
    * 発言前に `Thought` セクションを設け、キャラクターの「知識バイアス」や「姉妹関係の判定」を論理的に行わせる。
3.  **動的コンテキスト（Dynamic Context）の導入**:
    * 「AI基地建設計画」などの現在進行系のプロジェクトをステータスとして定義。

---
